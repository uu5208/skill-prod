# skill-prod

技能生产仓库 - 用于管理和部署 Agent Skills

## 目录结构

```
skill-prod/
├── skills/           # 技能目录
│   ├── skill-name/   # 每个技能一个目录
│   │   ├── SKILL.md  # 技能定义文件
│   │   └── references/ # 参考文档（可选）
│   │   └── templates/  # 模板文件（可选）
├── README.md         # 本文件
```

## 技能管理

- **新增技能**: 在 `skills/` 目录下创建新文件夹，包含 SKILL.md
- **更新技能**: 修改对应技能的 SKILL.md 文件
- **部署技能**: 通过 Agent 系统安装技能
